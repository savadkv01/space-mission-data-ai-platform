# later phases (ingestion / transformation / serving). To use them, either:
#   - build a custom Airflow image (recommended), or
#   - set _PIP_ADDITIONAL_REQUIREMENTS on the airflow service (slower boots).
#
# Pin versions when promoting beyond local development.

apache-airflow-providers-apache-spark  # SparkSubmitOperator -> spark-master
