# Satellite Telemetry — Data Quality Rules

> **Track:** Simulation (post-MVP). Satellite telemetry is generated by the
> synthetic telemetry generator and retained as the streaming / anomaly-ML
> demonstrator (see ADR-09 in [docs/data-modeling/17-adr.md](../../docs/data-modeling/17-adr.md)).
> These rules are documented for completeness and to keep the streaming track
> production-credible; they are **not** on the MVP critical path.

Entity: `silver_telemetry` (natural key `satellite_id`, `event_ts`).

---

## Mandatory fields

| Field | Type | Rule |
|-------|------|------|
| `event_ts` | ISO-8601 UTC | Non-null, parseable, not > 48h future / > 10y old |
| `satellite_id` | string | Non-null (natural key component) |
| `sensor_type` | string | Non-null; controlled vocabulary |
| `health` | enum | `∈ {NOMINAL, ANOMALY, UNKNOWN}` |

## Optional fields

| Field | Type | Rule |
|-------|------|------|
| `battery_voltage_value` | float | Within engineering limits (see below) |
| `battery_voltage_unit` | string | `= V` |
| `battery_voltage_status` | string | `∈ {NOMINAL, WARN, CRITICAL}` |
| `anomaly_sensor_count` | int | `≥ 0` |
| `anomaly_sensors` | list[string] | Length equals `anomaly_sensor_count` |
| `label_anomaly` | bool | Consistent with `health = ANOMALY` |

## Accepted ranges (engineering limits)

| Signal | Nominal | Warn | Critical |
|--------|---------|------|----------|
| Battery voltage (V) | 24.0 – 33.6 | 22.0 – 24.0 | < 22.0 or > 34.0 |

## Referential integrity, duplicates, timestamp, geospatial

- **Duplicate rule:** `(satellite_id, event_ts)` unique after dedup.
- **Timestamp rule:** telemetry per satellite must be non-decreasing in
  `event_ts`; regressions indicate clock skew or out-of-order replay.
- **Integrity rule:** each satellite belongs to exactly one mission (enforced in
  Gold business rules); `anomaly_sensors` must reference known sensor IDs.
- **Geospatial:** N/A for telemetry (see orbit rules for position).

**Business impact of failure:** corrupt telemetry biases `fact_sat_health`
health scores and the space-weather-to-anomaly correlation.
